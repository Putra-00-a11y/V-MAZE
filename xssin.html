Level 0 - Basic Alert

Deskripsi: Mulai dengan alert sederhana untuk memahami cara kerja XSS.

Requirement:

  • Browser modern
  • Pengetahuan dasar JavaScript

Skill yang harus dikuasai:

  • Dasar script injection
  • Syntax JavaScript dasar

Level 1 - Input Reflected

Deskripsi: Eksploitasi input yang direfleksikan untuk menjalankan payload XSS.

Requirement:

  • Browser modern
  • Pengetahuan dasar form input

Skill yang harus dikuasai:

  • Manipulasi Input
  • Konsep Reflected XSS

Level 2 - Stored XSS

Deskripsi: Payload yang tersimpan permanen di server dan dieksekusi pada setiap akses.

Requirement:

  • Browser modern
  • Server atau environment penyimpanan input

Skill yang harus dikuasai:

  • Konsep Stored XSS

Level 3 - URL Based XSS

Deskripsi: Eksploitasi URL tanpa server side, hanya di sisi client.

Requirement:

  • Browser modern
  • Pengetahuan URL dan Manipulasi Dasar

Skill yang harus dikuasai:

  • URL Traversal dan Manipulation
  • Event Handling Payload via URL

Level 4 - Filter Bypass Basic

Deskripsi: Pahami filter input sederhana dan coba bypass dengan teknik dasar seperti URL encoding atau penggunaan tanda kutip alternatif.

Requirement:

  • Browser modern
  • Pengetahuan dasar input sanitasi

Skill yang harus dikuasai:

  • URL encoding
  • Penggunaan alternatif karakter

Level 5 - Payload Intermediate

Deskripsi: Latihan membuat payload yang sedikit kompleks tapi masih mudah dimengerti, seperti chaining sederhana dan event handler injection.

Requirement:

  • Browser modern
  • Pengetahuan JavaScript dasar dan event

Skill yang harus dikuasai:

  • Event Handling/Event Listener payload
Mulai Challenge

  • Browser modern
  • Pengetahuan dasar web security

  • Analisis aplikasi web sederhana
